삼각법

피타고라스의 정리
a^2 + b^2 = c^2

삼각비 : 직각삼각형 변들의 비율
1:1:√2  3:4:5  5:12:13


삼각함수 : 인풋 값으로 세타를 갖고 아웃 풋값으로 삼각비를 갖는 함수
sinθ = b/c
cosθ = a/c
tanθ = b/a

a = cosθ * c
b = sinθ * c


디그리(dgree) 각도 : 일상생활에서 사용하는 각도
호도법 : 호의 길이를 사용해서 각도를 나타내는 방법

360도 -> 2ㅠ rad
1도 몇 라디안? ㅠ/180
70도는 몇 라디안? ㅠ/180 * 70

1라디안은 몇 도? 180/ㅠ
